Function va Date

Function

Ma'lum bir vazifani bajarish uchun yozilgan kodlar to'plamidir. Funksiya qayta-qayta chaqirilishi mumkin, shuning uchun ularni kodni takrorlashdan saqlanish uchun ishlatish juda foydalidir.

  • Funksiya tuzilishi:

    function functionNomi(parameter1, parameter2) {

    // body

    return result;

    }

  • Funksiyani chaqirish:

    function greet() {

    console.log("Hello, World!");

    }

    greet(); // "Hello, World!" chiqaradi

  • Parametrlar va Argumentlar:

    function sum(a, b) {

    return a + b;

    }

    console.log(sum(5, 3)); // 8

Date

JavaScript'da Date obyekti vaqt va sanani bilan ishlash uchun juda muhim bo'lib, u bizga hozirgi vaqtni olish, vaqtlar orasidagi farqlarni hisoblash, sanalarni formatlash va ko'plab boshqa amallarni bajarish imkonini beradi.

1. Hozirgi sanani olish: Bu usul hozirgiVaqt nomli o'zgaruvchiga joriy vaqtni beradi.

let hozirgiVaqt = new Date();

2. Aniq vaqtni belgilash: Oylar 0dan boshlanadi

let tugilganKun = new Date(2000, 5, 15); // Iyun 15, 2000

3. Vaqt va sanani string ko'rinishida kiritish:

let customDate = new Date("2023-11-05T14:30:00");

Asosiy metodlar

Yil, oy, kun va soatlarni olish:

let yil = hozirgiVaqt.getFullYear(); // yilni olish

let oy = hozirgiVaqt.getMonth(); // oy (0-11)

let kun = hozirgiVaqt.getDate(); // oy kunini olish

let soat = hozirgiVaqt.getHours(); // soatni olish

Yil, oy, kun va soatlarni o'rnatish:

hozirgiVaqt.setFullYear(2025); // yilni 2025 ga o'zgartirish

hozirgiVaqt.setMonth(11); // oy ni o'zgartirish (0-11)

hozirgiVaqt.setDate(25); // kunni o'zgartirish

hozirgiVaqt.setHours(10); // soatni o'zgartirish
